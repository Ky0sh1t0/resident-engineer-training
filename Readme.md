# Resident Engineer Training - C# Console System

Сборник задач над которыми я работал с прогрессией, при помощи C# и .NET

## Стэк технологий

- **Язык:** C#
- **Платформа:** .NET
- **Тип приложений:** Console Application

**Подход и концепция:**
- ООП (классы, объекты, свойства, методы)
- Работа с коллекциями(`List<type>`)
- Валидация ввода
- Обработка ошибок
- Управление состоянием объектов
- Архитектура консольных приложений

## Реализованные кейсы

### Case 01 - Access Control 

Базовое консольное приложение для ввода и отображения данных пользователя

**Реализовано:**
- Ввод имени, возраста, любимой игры и сколько часов проводит в неделю за играми
- Преобразования строки в число
- Вывод данных

**Освоенные навыки:** Переменные, Условные операторы (if, else), Базовая логика

---

### Case 02 - Access Control System

Система контроля доступа пользователя

**Реализовано:**
- Проверка возраста пользователя
- Проверка уровня доступа
- Определение зоны доступа

**Освоенные навыки:** `switch`

---

### Case 03 - Incident Archive Manager

Интерактивная система управления архивом инцидентов

**Реализовано:**
- Добавление записей
- Просмотр списка инцидентов
- Удаление по номеру
- Циклическое меню приложения
- Безопасная работа с индексами
- Обработка пользовательского ввода

**Освоенные навыки:** Циклы, `List<type>` и работа с ним, структура работы приложения

---

### Case 04 - Subject Registry

Объектно-ориентированная система управления сущностями

**Реализовано:**
- Создания класса `Subject`
- Свойства и конструктор
- Методы объекта
- Хранение объектов в `List<Subject>`

**Освоенные навыки:** ООП, работы со списками и объектами

--- 

### Case 05 - Status System

Система управления статусом сущностей из Case 04

**Реализовано:** 
- Добавление поля `status`
- Изменение статуса сущности через систему управление

**Освоенные навыки:** Управление состоянием объекта, изменение данных, архитектура интерфейса
